# Prabanandan Anandhamurugan - Portfolio

A professional portfolio website built with modern web technologies, showcasing the expertise and projects of a Java Software Developer.

## 🚀 Tech Stack

- **Framework:** React 19
- **Build Tool:** Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Components:** Shadcn/UI (Radix UI)
- **Icons:** Lucide React
- **Routing:** React Router DOM

## 📁 Project Structure

```text
src/
├── components/     # Reusable UI components (Hero, Skills, Experience, etc.)
├── pages/          # Main application pages (Index, Projects, NotFound)
├── lib/            # Utility functions and configurations
├── hooks/          # Custom React hooks
└── App.tsx         # Main application entry and routing
```

## 🛠️ Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run Development Server:**
   ```bash
   npm run dev
   ```

3. **Build for Production:**
   ```bash
   npm run build
   ```

## 📄 Features

- **Responsive Design:** Fully optimized for mobile, tablet, and desktop.
- **Dynamic Routing:** Seamless navigation between About and Projects pages.
- **Modern UI:** Clean aesthetics using Tailwind CSS and Shadcn/UI.
- **Interactive Elements:** Smooth transitions and hover effects.

To get a local copy of this project:
